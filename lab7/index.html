I looked into it myself at first, but didn't understand a lot of what was going on, so I asked WesBot/ChatGPT to help me out. It explained that it created a custom sorting method. It essentially checks what the array item WOULD be if toLowerCase was ACTUALLY run, without actually running it. When it does this, it can accurately sort the array items without changing the case, because it already recognized uppercase letters as what their lowercase letters would be.
